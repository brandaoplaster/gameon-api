# Game On API :video_game:

> An e-commerce for online game sales

## About the Game On project
Features present in this e-commerce:

 - Administrative area
 - Shopping cart
 - List of desired products
 - Shopping list
 - Product categorization
 - Search by filters
 - List of products
 - Payment integration

## Stack
 - Ruby 2.7.1
 - Rails 6.0.3.3
 - Postgres
 - Docker
 - Docker compose
